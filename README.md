# Getting Started with Application

First of all you have to install [git](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git), [NodeJS](https://nodejs.org/en/download/) > 14.17.1
then run the 
```sh
npm install
``` 
to download all project library dependencies

## Launch the project
Launch the project via 
```sh
npm run start
```
or navigate to `package.json` and do it manually with IDE terminal
